Overview

This document outlines the frontend changes needed to allow users to create and view project spaces in AppFlowy.

How It Works

Creating a Project Space

  1. User clicks "New project space" in sidebar
  2. Modal opens with:
    • Name field (required, max 100 chars)
  • Description field (optional, max 500 chars)
  • Icon selector (emoji picker)
  • Template choice (Kanban or Scrum)
    1. On submit:
      • Calls backend API to create space
  • Creates default board based on template
  • Navigates to the new board
  • Updates sidebar list

    • Viewing Project Spaces

    1. Project Spaces section in sidebar shows all spaces
    2. Each space displays:
      • Icon and name
  • Space type badge
  • Click to navigate to primary board
    1. Collapsible section with persistence
